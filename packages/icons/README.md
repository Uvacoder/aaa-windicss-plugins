# @windicss/plugin-icons

Generate on-demands icon library for [windicss](https://windicss.org/), powered by [css.gg](https://css.gg/app).

## Installation

```bash
npm install --save-dev @windicss/icons
```

## Usage

```js
// windi.config.js

plugins: [
  require('@windicss/icons'),
]
```

### Define icon size

* .icon-sm
* .icon-md
* .icon-lg
* .icon-xl
